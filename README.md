# 📱 iPhone Store

Современный интернет-магазин техники Apple.

## Возможности

* Каталог iPhone
* Просмотр характеристик устройств
* Удобный поиск товаров
* Актуальные цены
* Добавление товаров и удаление
* Интеграция с Telegram-ботом
* Адаптивный дизайн для смартфонов

## Технологии

* Python
* Django
* HTML/CSS
* JavaScript
* SQLite

## Установка

```bash
git clone https://github.com/USERNAME/REPOSITORY.git
cd REPOSITORY
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
