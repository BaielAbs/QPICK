📱 iPhone Store

Современный интернет-магазин техники Apple.

Возможности

* Каталог iPhone
* Просмотр характеристик устройств
* Удобный поиск товаров
* Добавление товаров и удаление
* Интеграция с Telegram-ботом
* Адаптивный дизайн смартфонов

Технологии

* Python
* Django
* HTML/CSS
* JavaScript
* SQLite

Установка

```bash
git clone https://github.com/USERNAME/REPOSITORY.git
cd REPOSITORY
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

О проекте

Проект создан для удобного просмотра и выбора техники Apple через сайт и Telegram-бота.
